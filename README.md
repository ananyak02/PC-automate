This code automates the capturing of point clouds from FARO cameras attached in Arena2036.
Two IP addresses are available 
camera 1 - https://141.70.213.50/
camera 2 - https://141.70.213.51/

The source code in written in python file backend.py (Flask + Selenium) and the frontend is developed using HTML / JS and wrriten in index.html

To run the code follow the instructions, first activate the environment, if there is no environment a requirements.txt file is provided in order to create a new environment. 

```
source pcautomate/bin/activate
```
